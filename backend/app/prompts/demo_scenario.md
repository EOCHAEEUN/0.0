# 팩토핏 데모 시나리오

## 시나리오 1 — 박 차장 (대시보드 진입)

**배경**
안산시 금속가공업체 생산팀장 박 차장. 15년 된 프레스가 잦은 고장에 에너지비도 높아서 교체를 고민 중.

### Step 1. 온보딩
```
기업명: 안산금속(주)
업종: 금속가공업 (C24)
지역: 경기도 안산시
직원 수: 45명
연간 에너지비용: 4,800만원
```

### Step 2. 대시보드 진입
- KPI 카드에 "연간 에너지비용 4,800만원 — 업종 평균 +38%" 빨간색으로 표시
- 설비 현황 카드에 "유압 프레스 라인 A — 15년 — 교체 권고" 표시
- "분석 시작 →" 버튼 클릭

### Step 3. ROI 분석 페이지
```
설비명: 유압 프레스 라인 A
설비 종류: press
설비 연령: 15년
연간 에너지비용: 4,800만원
불량률: 3.2%
```
→ "ROI 분석 시작" 클릭

### Step 4. 결과 카드
- 시나리오 A 카드: 실 기업 부담 5,600만원 / 회수기간 2.6년 / ROI 39%
- AI 추천 배지 표시
- "시나리오 A 신청서 초안 생성 →" 클릭

### Step 5. 신청서 초안 페이지
- 기업명, 신청 목적, 도입 설비, 기대 효과 자동 작성
- 추천 지원사업: KIAT 스마트 제조혁신 (D-42), 에너지공단 노후설비교체 (D-67)
- PDF 저장

## 시나리오 2 — 이 대리 (채팅 진입)

**배경**
시흥 산업단지 플라스틱 성형업체 총무팀장 이 대리. 올해 의무 안전점검 일정을 파악 못 한 상태.

### Step 1. AI 어드바이저 채팅 입력
```
"플라스틱 성형업체인데 올해 안전점검 뭐 남았어요?"
```

### Step 2. 에이전트 응답
- ✅ 소방설비 정기점검 — 완료 (2월)
- ⚠️ KTL 전기안전 정기검사 — D-67 / 미이행 시 최대 300만원 과태료
- ❌ KOSHA 화학물질 취급 안전점검 — 미확인 / 최대 500만원 과태료

### Step 3. 후속 질문
```
"KC인증 갱신도 확인해줘"
```

### Step 4. 에이전트 응답
- KC 인증 유효기간: 2027년 3월 (9개월 남음)
- 갱신 준비 권장 시작: 2026년 12월
- KTL 갱신 심사 평균 2~3개월 소요 안내
